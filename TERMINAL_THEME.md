# Terminal Theme Configuration

For the best visual experience with lazysetup, configure your terminal with the following color scheme:

## Recommended Colors

- **Background**: Dark Navy/Charcoal (#2d2d44 or similar)
- **Foreground**: White/Light Gray
- **Magenta (Highlight)**: Bright Magenta (#ff00ff or similar)
- **Cyan (Accent)**: Bright Cyan (#00ffff or similar)

## Terminal Configuration

### macOS (Terminal.app)
1. Open Terminal Preferences
2. Go to Profiles tab
3. Create or select a profile
4. Set background color to dark navy (#2d2d44)
5. Set text color to white
6. Configure ANSI colors with bright magenta and cyan

### macOS (iTerm2)
1. Open iTerm2 Preferences
2. Go to Profiles â†’ Colors
3. Import or create a color preset with:
   - Background: #2d2d44
   - Text: White
   - ANSI Magenta: Bright Magenta
   - ANSI Cyan: Bright Cyan

### Linux (GNOME Terminal, Konsole, etc.)
Configure your terminal emulator's color scheme with the colors listed above.

## Note

The application uses the terminal's native color capabilities. Ensure your terminal supports 256-color or true color mode for best results.
